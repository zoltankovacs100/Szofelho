## Letöltés

A betűtípusokat a Google Fonts-ról vagy a Font Squirrel-ről lehet letölteni WOFF2 és WOFF formátumban.

## Használat

A betűtípusok automatikusan betöltődnek a `fonts.css` fájlban definiált `@font-face` szabályok alapján.
